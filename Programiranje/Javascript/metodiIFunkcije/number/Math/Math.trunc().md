#javascript #math #number #rounding #method #function 
## Opis
Odstranjuje sve posle decimalnog zareza.

## Sintaksa:
Math.trunc(number), gde:
- number je broj na kom radimo

## Nacin koriscenja:
```js
Math.trunc(3.1); // 3
Math.trunc(-3.1); // -3
```
