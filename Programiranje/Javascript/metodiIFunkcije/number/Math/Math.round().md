#javascript #math #number #rounding #method #function 
## Opis
Zaokruzuje na najblizi ceo broj. Ukoliko imamo .5, zaokruzuje nagore.

## Sintaksa:
Math.round(number), gde:
- number je broj koji zaokruzujemo

## Nacin koriscenja:
```js
Math.round(3.1); // 3
Math.round(-3.1); // -3
Math.round(3.5); // 4
```
